### AI Assignment (Summary 4) 

The following poses a viable option for an assingment:

1. Pathogenic_Protein_Interaction_Simulator

The user is given a default pathogenic amyloid protein to view its interactions with cellular componnents as well as modify other       environmental factors such as oxidative stress and receptor density. An AI-json file template is provided to allow the user to explore different interactions of the pathogenic (or not) protein with other cellular components that can be provided to an LLM or AI to fill out the protein "specs" of choice. This simulator is in its early stages and can be modified to include interactions with glial cells and dopaminergic cells to name a few. As such, the available iteration of this simulator is simply to see whether or not this assignment is a viable option. 


2. Pathogenic_Amino_Acid_Simulator

The user is able to manipulate the amino acid sequence of a pathogenic protein. Using AI, the user can experiment with different amino acid sequences to see the conformational changes of the protein.
