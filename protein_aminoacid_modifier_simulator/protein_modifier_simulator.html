Static HTML Workspace

Toxic Protein Sequence Viewer

Upload a FASTA file, paste amino-acid text, compare a clean reference against an edited sequence, and export the result as FASTA or a compact mutation report. This runs as a single HTML file with no framework and no backend, with presets for alpha-synuclein and amyloid-beta peptides.

Scope

This page is for browser-based sequence viewing and editing. It is not a clinical interpretation engine, a protein design system, or a substitute for validating the exact accession and isoform you plan to study.

ChatGPT workflow Ask ChatGPT for FASTA text, save it as a plain `.fasta` or `.txt` file, then upload it here for visualization and editing.
Repository fallback If you want live retrieval, this page can fetch a UniProt accession when opened from a local server such as VS Code Live Server.
